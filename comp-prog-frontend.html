Sass - CSS

Tout d'abord qu'est-ce que le CSS ? Cela permet de stylisé une page HTML par exemple pour ajouter de la couleur, grossir un titre etc.
Et Sass ? Sass est un langage de script préprocesseur qui est compilé ou interprété en CSS, c'est en quelques sorte une extension du CSS.

Durant mon cursus à INTECH, nous codions en CSS pure, j'ai remarqué que plus mes projets étudiants grandissait en terme de base de code, plus le code se répétait et plus il était difficile de le maintenir.
Vint mon stage chez Nodata et j'ai découvert une amélioration au CSS que je nomme Sass et depuis je ne jure que par lui.
